## SQLmap instalatu

1. Sistema eguneratu
   ```bash
   $ sudo apt update
   ```
2. SQLmap instalatu
   ```bash  
   $ sudo apt install sqlmap
   ```
3. Instalazioa egiaztatu
   ```bash
   $ sqlmap --version
   ```
